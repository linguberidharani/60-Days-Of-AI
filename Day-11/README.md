# Day 11 – ATS Resume Optimizer & Resume Generator

## Objective

Learn how Applicant Tracking Systems (ATS) evaluate resumes and use Claude AI to optimize a resume for better recruiter visibility while maintaining complete factual accuracy.

---

## Prompt Used

### ATS Resume Optimizer & Resume Generator

**Input:**

* Existing Resume
* Target Job Description

**Task:**
Rewrite the resume for maximum ATS alignment and recruiter relevance while maintaining 100% factual accuracy.

**Rules:**

* Never invent experience, projects, employers, certifications, dates, metrics, or skills.
* Only optimize, reorganize, and rephrase existing content.
* Use relevant JD keywords naturally.
* Keep ATS-friendly formatting.

**Output:**

1. ATS Match Score
2. Gap Analysis

   * Missing Keywords
   * Missing Skills
   * Improvement Opportunities
3. Updated Resume

## Key Learnings

1. ATS systems prefer standard resume structures.
2. Keywords significantly impact resume visibility.
3. Professional Summary performs better than Career Objective.
4. Single-column layouts improve ATS parsing.
5. Action-oriented bullet points increase recruiter readability.
6. GitHub and project visibility improve technical profile strength.
7. ATS optimization should improve readability without exaggerating skills or experience.

---

## Biggest Insight

A resume should be optimized for both:

* Applicant Tracking Systems (ATS)
* Human Recruiters

Small improvements in formatting, keyword alignment, and structure can significantly increase interview opportunities.
