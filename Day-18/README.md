# 🚀 Day 18 – Build a Brain Dump Action Planner Skill

## 📌 Objective

Today I learned how to build a reusable Claude Custom Skill that transforms unstructured notes, meeting transcripts, brainstorming sessions, and voice memos into organized project dashboards and actionable plans.

Instead of manually reviewing lengthy notes, the AI automatically extracts summaries, action items, risks, blockers, decisions, and open questions into a clean interactive HTML dashboard.

---

# 🧠 What I Learned

- Creating reusable AI workflows using Claude Custom Skills
- Transforming messy notes into structured dashboards
- Organizing meeting transcripts automatically
- Extracting action items, risks, and decisions using AI
- Building responsive HTML dashboards from unstructured information
- Improving productivity through reusable prompt engineering

---

# ⚙️ Custom Skill Created

**Skill Name**

brain-dump-action-planner

### Purpose

Convert raw notes, meeting transcripts, brainstorming sessions, and voice memos into a structured interactive dashboard without manually organizing information.

- Executive Summary
- Key Takeaways
- Speaker-wise Summary
- Action Items Dashboard
- Risks & Blockers
- Open Questions

- Attribution Notes
- Interactive HTML Output
- Responsive Dashboard Layout
