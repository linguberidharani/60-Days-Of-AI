# 🚀 Day 20 – Build an AI Face Puzzle Game

## 📌 Objective

Today I built an interactive **AI Face Puzzle Game** using Claude AI. The application captures a user's photo through the webcam and transforms it into a fun, responsive puzzle game with multiple difficulty levels, timer tracking, and a local leaderboard.

---

## 🧠 What I Learned

- AI-assisted browser game development
- Camera integration using the MediaDevices API
- Image processing with the HTML5 Canvas API
- Puzzle generation and game logic
- Drag-and-drop and touch interactions
- Timer, move counter, and progress tracking
- LocalStorage for leaderboard management
- Responsive UI design using HTML, CSS, and JavaScript

---

## 🎮 Features

- 📷 Live Webcam Preview
- 🤳 Capture Face using Camera
- 🧩 Puzzle Generation (3×3, 4×4, 5×5)
- 🖱️ Drag-and-Drop & Touch Controls
- ⏱️ Live Timer
- 🎯 Move Counter
- ✅ Correct Pieces Tracker
- 🏆 Win Detection
- 📊 Leaderboard with LocalStorage
- 🔄 Retake Photo & Play Again
- 📱 Responsive Design
- 🌙 Dark/Light Theme
- 🎉 Smooth Animations & Celebration Effects
