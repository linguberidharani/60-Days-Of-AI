# Day 9 – Build & Enhance an AI Nutrition Analytics App

## Topic
AI Product Building using Iterative Development

## Objective
Build a Nutrition Analytics Application called NutriScope and then enhance it using a second prompt.

---

## Prompt 1 – MVP Version

Built a complete HTML nutrition dashboard with:

- User profile inputs
- Food logging
- Nutrition tracking
- Macro & micronutrient calculations
- Progress dashboard
- Recommendations engine

### MVP Features

- Age, Gender, Height, Weight
- Activity Level
- Dietary Preference
- Food Logging
- Nutrition Dashboard
- Charts
- Recommendations

---

## Prompt 2 – Enhanced Version

Enhanced the NutriScope application by adding:

- CSV Upload
- 40+ Additional Foods
- Risk Analysis
- Meal Planner
- Advanced Recommendations
- Nutrition Sources
- Better Visualizations
- Educational Disclaimer

---

## Comparison

### MVP Version

- Simple nutrition tracking
- Basic recommendations
- Core dashboard

### Enhanced Version

- Advanced analytics
- Risk assessment
- Meal planning
- Larger food database
- Improved user experience

## Key Learnings

1. Build MVP first before adding complexity.
2. Iterative prompting produces higher quality applications.
3. AI can generate fully functional dashboard applications.
4. Breaking large applications into smaller stages improves reliability.
5. Enhancement prompts are powerful for product development.
