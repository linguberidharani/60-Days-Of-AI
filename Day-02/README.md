# Day 2 - Prompt Engineering

## Objective

Learn the difference between a normal prompt and an engineered prompt.

## What I Learned

Prompt Engineering is the skill of communicating effectively with AI to obtain better outputs.

A well-structured prompt can:

* Improve accuracy
* Reduce hallucinations
* Produce expert-level responses
* Save time by reducing iterations

## Prompt Comparison

### Lazy Prompt

```text
Create an image explaining Prompt Engineering
```

### Engineered Prompt

```text
Act as an expert AI educator and infographic designer.

Create a professional infographic explaining Prompt Engineering for beginners.

Requirements:
- Title: What is Prompt Engineering?
- Explain the concept in simple language
- Include why prompt engineering matters
- Show the difference between a lazy prompt and an engineered prompt
- Include 4 benefits:
  - More accurate answers
  - Expert-level outputs
  - Faster workflows
  - Reduced hallucinations
- Use a modern AI-themed visual style
- Organize content with clear sections and examples
```

## Key Observations

* The lazy prompt produced a generic output.
* The engineered prompt produced a more structured and informative result.
* Adding role, context, objectives, and constraints significantly improved output quality.

## Key Takeaway

Prompt Engineering is not about asking more questions. It is about asking better questions with clear context, structure, and goals.
