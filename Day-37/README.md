# Day 37 – Task Compass 🧭

## Overview

Today I built **Task Compass**, an interactive management simulation that teaches how work flows through real organizations.

Instead of testing job titles, the application focuses on understanding:

- Ownership
- Delegation
- Collaboration
- Workflow Thinking

For this project, I selected the **Freelancer / Small Business** workplace and explored how different roles work together to solve real business scenarios.

---

## Features

### Stage 1 – Who Owns This?
- Drag-and-drop role assignment
- Real-world workplace tasks
- Primary owner identification
- Explanation of ownership
- Supporting roles for each task

### Stage 2 – Task Routing
- Build complete organizational workflows
- Arrange roles in the correct sequence
- Animated workflow visualization
- Workflow reasoning after each challenge

### Stage 3 – Collaboration Challenge
- Assign multiple teams to complex situations
- Learn how departments collaborate
- Communication flow visualization
- Practical organizational reasoning

### Final Dashboard
- Understanding Dashboard
- Ownership score
- Delegation score
- Collaboration score
- Workflow Thinking score
- Personalized reflection and insights

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Native Drag & Drop API

No frameworks or external libraries were used.

The application works completely offline.

---

## Key Learnings

- Ownership is about assigning responsibility to the most appropriate role.
- Delegation ensures work reaches the right people efficiently.
- Most workplace challenges require collaboration rather than a single owner.
- Organizational workflows follow structured communication and decision-making processes.
- Understanding how work moves across teams is just as important as completing individual tasks.
