# Day 39 – PDF Splitter & Merger

## 📌 Challenge
Build **PDF Splitter & Merger** – Create a Premium Browser-Based PDF Utility.

## 🎯 Objective
Develop a professional browser-based PDF utility that allows users to split and merge PDF files completely on the client side without uploading documents to any server.

## 🚀 Features Implemented

### 📄 PDF Splitter
- Upload PDF documents
- Automatic page detection
- Page thumbnail preview
- Multiple split methods:
  - Single Page
  - Custom Range
  - Multiple Ranges
  - Every N Pages
  - Split After Page
  - Extract Selected Pages
  - Odd Pages
  - Even Pages
  - First N Pages
  - Last N Pages
  - Reverse Order
- Queue multiple split operations
- Download processed PDFs

### 📑 PDF Merger
- Upload multiple PDFs
- Drag & Drop support
- File reordering
- Page count and file size display
- Merge multiple PDFs into a single document
- Download merged PDF

### 🎨 UI/UX
- Premium dark theme
- Glassmorphism interface
- Responsive design
- Smooth animations
- Processing indicators
- Success dialogs
- Modern dashboard

## 🛠️ Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
- Client-side PDF processing libraries
