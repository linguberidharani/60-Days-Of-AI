# Day 12 – Build Your Complete Job Search & Personal Branding Toolkit

## Objective

Learn how Claude can be used as a Career Coach, Recruiter, Resume Writer, and Personal Branding Consultant to create a complete job search toolkit.

---

## Prompt Used

```text
You are an expert Technical Recruiter, Hiring Manager, Career Coach, Executive Resume Writer, and Personal Branding Consultant.

Carefully analyze my resume, LinkedIn profile, portfolio, and professional background.

Your objective is to create a complete job search and personal branding toolkit tailored to my profile.
```

(Full prompt provided in challenge resources.)

11. Personal Brand Summary

---

## Key Learnings

- Personal branding is more than a resume.
- Recruiter outreach messages can be AI-assisted.
- A strong positioning statement improves visibility.
- Skill gap analysis helps prioritize learning.
- Career assets should be ATS-friendly and recruiter-focused.
