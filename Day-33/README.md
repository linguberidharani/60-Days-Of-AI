# Day 33 – Media Integrity Analyzer

## Overview

Built an interactive Media Integrity Analyzer that teaches users how to identify misleading headlines, emotional manipulation, and media bias through guided learning and interactive challenges.

## Features

- Theme Selection
- Headline Detective Challenge
- Emotion Detector Challenge
- Live Media Integrity Metrics
- Headline Accuracy Analysis
- Fair Headline Rewrite
- Emotion Analysis
- Final Media Integrity Dashboard
- Replay with New Scenarios
- Responsive Dark UI

## Technologies Used

- HTML
- CSS
- JavaScript

## What I Learned

- Media literacy concepts
- Detecting misleading headlines
- Recognizing emotional manipulation
- Interactive UI design
- JavaScript-based state management
- Building responsive single-page applications
