# Day 31 – AI Supply Chain Control Tower

## Overview
Built an interactive AI Supply Chain Control Tower that simulates managing supply chain disruptions. Players make decisions to maintain business performance while handling real-time alerts.

## Features
- Live KPI Dashboard
- Real-time Alerts
- Decision-based Gameplay
- Countdown Timer
- Event Log
- Pause & Help Options
- Final Score Dashboard
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript

## What I Learned
- Building interactive dashboards
- Managing dynamic data with JavaScript
- Creating responsive user interfaces
- Implementing game logic and scoring
