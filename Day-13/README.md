# Day 13 – Build Your AI Job Search Assistant

## Objective

Learn how Claude Connectors can be used to automate job discovery, analyze opportunities, identify skill gaps, and create a personalized job search strategy.

---

## Activities Completed

- Connected Indeed with Claude
- Created a professional profile
- Defined job search preferences
- Generated matching job opportunities
- Reviewed match scores
- Analyzed required skills
- Studied market demand insights
- Identified learning priorities

---

## Market Insights

- Cybersecurity demand continues to grow globally.
- Security Analyst and SOC roles remain strong entry-level opportunities.
- Practical experience is highly valued.
- Hands-on labs and projects significantly improve employability.

---

## Key Learnings

1. AI can automate job discovery.
2. Match scores help prioritize applications.
3. Skill-gap analysis provides a focused learning roadmap.
4. Market demand data helps guide career planning.
5. Networking and personal branding remain important alongside technical skills.
