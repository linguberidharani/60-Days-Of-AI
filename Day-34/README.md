# Day 34 – Marketing Detective

## Overview

Built an interactive Marketing Detective web application that allows users to investigate fictional marketing campaigns, analyze evidence, identify campaign mistakes, and learn practical marketing strategies through a detective-style experience.

---

## Features

- Theme Selection
- Interactive Case Assignment
- Investigation Board
- Clue Analysis
- Solve the Marketing Case
- Case Closed Animation
- Learning Report
- Campaign Performance Charts
- Marketing Investigation Certificate
- Replay with Random Marketing Cases
- Responsive Dark Detective Interface

---

## Technologies Used

- HTML
- CSS
- JavaScript

---

## What I Learned

- Marketing campaign analysis
- Audience targeting strategies
- Marketing channel evaluation
- Budget allocation analysis
- Interactive UI/UX design
- Building single-page web applications
